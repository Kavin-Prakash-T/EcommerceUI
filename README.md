# E-commerce Website

A simple e-commerce website built with HTML, CSS, and JavaScript.

## Project Structure

```
├── index.html          # Home page
├── collection.html     # Products collection page
├── contact.html        # Contact page
├── index.js           # Main JavaScript file
├── collection.js      # Collection page JavaScript
├── style.css          # Main stylesheet
└── img/               # Image assets
    ├── icons/         # Icon images
    ├── others/        # Miscellaneous images
    └── products/      # Product images
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the site navigation

## Features

- Home page with product showcase
- Product collection/catalog page
- Contact page
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
